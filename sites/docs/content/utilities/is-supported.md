category: Utilities
---

## Usage

```svelte
<script lang="ts">
	import { IsSupported } from "runed";

	const isSupported = new IsSupported(() => navigator && "geolocation" in navigator);

	if (isSupported.current) {
		// Do something with the geolocation API
	}
</script>
```
